Pixelite - Pixel art app using React
===================

This 'pixelite' app is a paint-by-numbers pixel art app built with React and TypeScript, styled with Tailwind CSS, and bundled with Vite. It allows users to create pixel art by filling in a canvas with colors from a palette, following a paint-by-numbers style guide.

Explore the app live at: https://pixelite.pages.dev/
